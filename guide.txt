# guide

# set colour palette
1 -> set multicolour
2 -> set colour red
3 -> set colour orange
4 -> set colour yellow
5 -> set colour green
6 -> set colour blue
7 -> set colour violet
8 -> set colour cyan
9 -> set colour gray

# set background colour
H -> previous background colour 
J -> next background colour

# pen state
Spacebar -> clear screen
Z -> draw background
X -> draw square (fill)
C -> draw square (outline)
V -> draw circle (fill)
B -> draw circle (outline)
N -> draw square/circle (fill)
M -> draw square/circle (outline)

# movement
Up -> move up
Down -> move down
Left -> move left
Right -> move right

# movement speed
W -> speed up
S -> speed down

# set size
A -> size down
D -> size up

# set line size
E -> line size down
R -> line size up

# set strobe on/off
O -> strobe on
P -> strobe off

# if strobe off, traverse shade
F -> shade up
G -> shade down

# close
Escape -> exit
